# Thyroid Cancer Risk Prediction

This project uses a deep feedforward neural network to predict thyroid cancer risk based on a dataset. The model aims to achieve:
- Accuracy: 82.5%
- Recall: 44.68%
- Precision: 69.38%
- F1 Score: 54.36%
